# MySuperTour Search v2.2.0

ГОРОДА = parent product_cat.  
РУБРИКИ = дочерние product_cat выбранного города.  
ПРОДУКТЫ = WooCommerce product.  
ПОЛЕЗНАЯ ИНФО = post.  

СИНОНИМЫ: пориж=париж, paris=париж, brussels=брюссель и т.д.  
ТРАНСЛИТ: Paris ↔ Париж, Brussels ↔ Брюссель.  
Fuzzy: расстояние Левенштейна ≤ заданного.  
Иконки: term meta msts_icon (эмодзи / текст / URL).  

Шорткод: `[mst_search]`.

Установка: скопировать папку, активировать, настроить.
